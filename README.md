# ai-upgrade-path
A Python project for delivering digestible content with practical exercises.

## Features
- Module length metadata shows 5-10 minutes
- Each module includes a 2-step practical exercise
- Content is rendered in responsive UI with dark/light mode
- Progress bar updates after each exercise step

## Usage
1. Create a new module using the `create_module` function
2. Render the module content using the `render` method
3. Update the progress bar using the `update_progress` method

## Testing
Run the tests using `python -m pytest`
